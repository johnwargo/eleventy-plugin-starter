## The `sayHello` Filter 

This filter generates HTML from my first name:

| Style  | Results                     |
| ------ | --------------------------- |
| H1     | {{ firstName | sayHello: "h1" }} |
| H2     | {{ firstName | sayHello: "h2" }} |
| H3     | {{ firstName | sayHello: "h3" }} |
| H4     | {{ firstName | sayHello: "h4" }} |
| p      | {{ firstName | sayHello: "p" }}  |

## The `commaize` Filter
